Conjunto HTML5, CSS e JS

HTML

é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

é uma linguagem de definição de estilo.

Javascript

é uma linguagem de programação interpretada.

Tim Berners-Lee criou o HTML original (e outros protocolos associados como o HTTP) para resolver um problema: a comunicação e disseminação das pesquisas entre ele e seu grupo de colegas. Sua solução, combinada com a então emergente Internet ganhou atenção mundial.
